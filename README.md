# Tic Tac Toe Game

A simple web-based **Tic Tac Toe game** built using HTML, CSS, and JavaScript.  
Users can play against the computer with a responsive game board.

## Features
- Play Tic Tac Toe against the computer
- Interactive 3x3 grid with X and O marks
- Detects winning, losing, or draw conditions
- Reset option to start a new game
- Responsive design for mobile and desktop

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Use
1. Open `index.html` in your browser.
2. Click on a cell to place your mark (X).
3. The computer will make its move automatically.
4. The game detects wins, losses, or draws automatically.
5. Click **Reset** to start a new game.
